# Express File Manager App

## 📌 Project Description
This is a simple backend application built using Node.js and Express.
It allows users to create, rename, and manage text files using the File System (fs) module.
This project demonstrates basic CRUD (Create, Read, Update, Delete) operations in a backend environment using Express routing.

## 🚀 Features
- Create new text files
- Rename existing files
- Display file content dynamically
- Redirect user after operations
- Handle form data using POST requests

## 🛠️ Technologies Used
- Node.js
- Express.js
- File System (fs module)
- EJS

## 📂 Installation & Setup
1. Clone the repository  
   git clone https://github.com/msoyabcode/express-file-manager.git

2. Navigate to the project folder  
   cd express-file-manager

3. Install dependencies  
   npm install
   
4. Start the server  

   For development:
   npm run dev

   For production:
   npm start

5. Open in browser  
   http://localhost:3000
